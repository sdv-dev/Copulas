copulas.univariate.base module¶

class copulas.univariate.base.ScipyWrapper(*args, **kwargs)[source]¶

Bases: copulas.univariate.base.Univariate

Wrapper for scipy.stats.rv_continous subclasses.

This class is intended to be used to integrate random variables from scipy.stats into copulas. It contain 5 attributes that control its behavior:

  • model_class: Name of the class to integrate, it must be in the scipy.stats module.

  • probability_density, cumulative_distribution, percent_point, sample : This attributes contain the information about how to map the methods in the model.

    • If it’s of type str it will interpreted as the name of the corresponding method in model.

    • If it’s None it will be interpreted as the method doesn’t exist in the model and is not implemented and a error message will be displayed.

    • If it’s not present it will be interpreted as the method doesn’t exist in the model but has been implemented in the integration.

On fit time it will instantiate the given model_class name, and map its methods to the class, following the values in the attributes for the corresponding method.

You can subclass ScipyWrapper by:

  1. On a file named after your distribution, create a new subclass.

  2. Set the model_class with the name of the distribution you want to integrate.

  3. Map the methods of your model.

  4. Implement the from_dict and _fit_params methods.

  5. Implement any custom method not mapped.

For a working example of how to implement subclasses of ScipyWraper, please check the source of copulas.univariate.kde.
